This website is my final project for CS 5630 at the University of
Utah. It is a tool to explore the effectiveness and popularity of
chess openings. Data is pulled from lichess.com's public database of
online chess games. One can filter the games loaded into the
visualization by skill level. This visualization is intended to
answer questions about the relationship between player skill level,
move popularity, and move effectiveness. For more detail about the
project, see the attached Process Book!
